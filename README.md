# 🌐 Raj Roy's Personal Website

This repository contains the source code for my personal website. It is a minimal, fast, and fully client-side site built with **HTML, CSS, and JavaScript**, designed to be long-lasting and easy to maintain without frequent updates.

---

## 🚀 Overview

The website serves as a simple, elegant digital identity hub featuring:

- **Minimalist Typographic UI:** Focused on clean presentation and sharp layout semantics.
- **Interactive Expandable Bio:** A clean inline and block text reveal system to present an immersive reading experience.
- **Dynamic Age Calculation:** Automatically calculates and updates chronological age based on real-time client tracking.
- **Integrated Contact Hub:** An interactive email concealment slider featuring automated clipboard execution and temporary status toast feedback.

No backend, frameworks, or heavy build tools are required — everything runs directly in the browser with maximum performance.

---

## 🛠️ Built With

* **HTML5:** Semantic architecture layout.
* **CSS3:** Custom properties, responsive variables, and hardware-accelerated transitions.
* **Vanilla JavaScript:** Clean, event-driven DOM manipulation and native browser API utilization.
